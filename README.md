# tdtzj
本实例程序是基于vue-cli3.0+openlayer搭建的调用天地图浙江的基本框架，启动方式和方法都和vue-cli官网一样一样的。

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
